## 📖 项目简介

EduLens 是一款专为在线教学和学习场景设计的浏览器扩展插件。它提供了丰富的辅助工具，帮助教师和学生提升在线课堂体验，支持实时协作和多人同步标注。

### ✨ 核心特点

- 👥 **实时协作** - 多人房间系统，支持实时同步标注
- 💾 **云端同步** - 标注数据自动保存到本地和云端

- 🎨 **丰富的标注工具** - 涂鸦绘制、框选批注、书签、图片等多种标注方式
- 📸 **强大的截图功能** - DOM 截图、区域截图、长屏滚动截图
- 🎯 **阅读辅助** - 聚光灯、鼠标高亮、倒计时等专注工具
- 🌍 **国际化支持** - 内置中英文双语

### 👥 适用人群

- **教师群体** - 在线远程授课时进行实时标注、演示和互动
- **学生用户** - 网课学习中做笔记、标记重点、专注阅读
- **视频录制** - 录制教学视频时进行可视化标注

## 🎯 使用指南

**工具面板**

启动插件后，面板气泡会出现在网页右下角，可随意拖拽移动位置，点击展开工具面板。

### 📌 学习工具

- **滚动进度条** - 工具面板左侧竖轴实时显示阅读进度
- **书签功能** - 在想要标记的位置输入标题并添加书签；点击竖轴处的圆点书签可快速跳转位置
- **倒计时器** - 输入分钟数，点击进入倒计时，时间到达后会弹出提醒

### 🎨 标注工具

- **涂鸦绘制** - 点击"涂鸦笔"功能，选择颜色和笔刷大小，即可自由绘制；支持直线工具、橡皮擦和一键清除
- **框选批注** - 点击"框选批注"后在页面拖拽绘制矩形框，在框选中输入批注内容；悬停可查看完整内容
- **图片挂起** - 点击"图片"上传本地图片，可拖拽调整位置、大小和旋转角度，支持切换固定/滚动模式

### 📸 截图工具

- **DOM 截图** - 点击按钮后，悬停页面元素会高亮，点击即可截图该元素
- **区域截图** - 在页面拖拽绘制矩形区域，释放鼠标即可截图
- **滚动截屏** - 自动滚动页面并拼接完整长图，点击任意位置可提前结束

### 🔍 专注工具

- **鼠标高亮** - 快捷键 `Alt+H`，鼠标周围区域增强显示
- **聚光灯** - 快捷键 `Alt+S`，高亮中心区域，周围变暗
- **阅读聚光灯** - 快捷键 `Alt+R`，跟随阅读位置的聚光灯效果

### 👥 协作功能

> ⚠️ 协作功能需要连接服务器，未连接时自动使用本地模式（数据保存在 chrome 浏览器本地）

- **登录注册** - 点击"登录"按钮，或"注册"填写用户名、邮箱和密码（至少 6 个字符），注册成功后自动登录
- **创建房间** - 登录后点击"房间列表"内的"创建新房间"，填写房间名称和描述
- **分享房间** - 在”房间列表“中点击"分享"按钮，可复制分享码发送给他人
- **加入房间** - 点击"加入新房间"，输入收到的分享码即可加入
- **切换房间** - 在房间列表中点击"切换"按钮即可切换到其他房间
- **实时同步** - 加入同一房间并访问相同网页时，书签、涂鸦、框选批注和图片等操作会实时同步给所有成员
