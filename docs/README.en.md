## üìñ Overview

EduLens is a browser extension designed for both personal learning and online teaching.

 It offers real-time collaborative annotation, a variety of note-taking tools, and reading aids ‚Äî helping students organize knowledge efficiently while enhancing interactive classroom experiences for teachers.

### Core Features

- üë• **Real-time collaboration** - Multi-user rooms keep annotations in sync
- üíæ **Cloud sync** - Annotation data is stored locally and in the cloud
- üé® **Annotation toolkit** - Freehand drawing, box notes, bookmarks, inline images, and more
- üì∏ **Powerful screenshots** - Capture DOM nodes, custom regions, or full-page scrolling shots
- üéØ **Reading assistance** - Spotlight, cursor highlight, countdown timer, and other focus helpers
- üåç **Internationalization** - Built-in Simplified/Traditional Chinese, English, Japanese, German, and Spanish

### Use Cases

- **Self-Study** - Take notes, highlight key points, and use reading aids to maintain focus while learning online
- **Online Teaching** - Enable real-time multi-user annotation and interaction during remote lessons to enhance engagement
- **Video Production** - Add visual annotations when recording educational videos for clearer and more intuitive content

## üìù Usage Guide

After activation, a floating bubble appears in the bottom-right corner of any page. Drag to reposition, then click to open the toolbox.

#### Learning Tools

- **Scroll progress** - A vertical bar on the panel's left side shows how far you have read
- **Bookmarks** - Enter a title at any point to add a bookmark; click the dot on the bar to jump back
- **Countdown timer** - Input minutes and start; a reminder pops up when time is up

#### Annotation Tools

- **Freehand drawing** - Choose "Doodle," pick colors and brush size, draw freely, or switch to straight lines, eraser, and clear-all
- **Boxed notes** - Select "Box Annotation," drag a rectangle, and type the note; hover to preview the full text
- **Pinned images** - Upload local images, drag to reposition, resize, or rotate; lock them in place or let them scroll

#### Screenshot Tools

- **DOM capture** - Hover to highlight any element and click once to capture it
- **Area capture** - Drag a rectangle to take a custom screenshot
- **Scrolling capture** - Auto-scrolls and stitches the full page; click anywhere to stop early

#### Focus Tools

- **Cursor highlight** - Shortcut `Alt+H` brightens the area around the cursor
- **Spotlight** - Shortcut `Alt+S` lights up the center while dimming the surroundings
- **Reading spotlight** - Shortcut `Alt+R` follows your reading position

#### Collaboration Tools

> ‚ö†Ô∏è Collaboration requires a server connection. When offline, EduLens switches to local mode (data stays in Chrome storage).

- **Sign up & sign in** - Click "Login" or "Sign Up," provide username, email, and a 6+ character password; auto login after registration
- **Create rooms** - After signing in, open "Room List" and select "Create New Room," then add a name and description
- **Share rooms** - Click "Share" inside the room list to copy an invite code for teammates
- **Join rooms** - Choose "Join New Room," enter the invite code, and start collaborating
- **Switch rooms** - Move between rooms with the "Switch" button
- **Live sync** - When teammates share the same room and page, bookmarks, drawings, boxes, and images sync instantly
